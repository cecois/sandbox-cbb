underlying platform
OpenShift v3 was too much infrastructure - this thing is being migrated to a VPS for more hands-on management
processing
Believe it or not, there's kind of a lot of processing that goes on in order to generate this dumb thing. Data need to be pushed/pulled from a MongoDB instance, indexed in a Solr instance, then visualized, etc. The work behind the scenes is being rewritten in NodeJS for easier and faster everything.
design
Partially inspired by the show's own move into (and out of, kinda) Howl and then Stitcher, the design is undergoing a refresh, all the way down to its CSS framework. The result will be cleaner and simpler. It's for the best.
